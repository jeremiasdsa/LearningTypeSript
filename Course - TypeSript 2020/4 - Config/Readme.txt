Instructions


$npm init
will create a package.json file

$npm install --save-dev lite-server
will install lite-server lib


Edit package.json adding this line on script first part:
"start": "lite-server"

$npm start
to run the lite-server and access on local host
also to update automaticaly when any file is changed on the project.

$npm install 
to install all dependencies and recreate node_modules folder
